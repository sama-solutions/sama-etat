🏛️ Module 2 : Présider les assemblées ✅

🎯 Objectifs d'apprentissage

  • Planifier et organiser les assemblées générales
  • Présider efficacement les réunions
  • Gérer les votes et décisions
  • Assurer le suivi des résolutions

📅 Planification d'une Assemblée

📋 Étape 1 : Création de l'assemblée
  1. Accès : Menu Assemblées → Nouvelle Assemblée
  2. Type : Sélectionnez "Assemblée Générale Ordinaire" ou "Extraordinaire"
  3. Date et lieu : Définissez la date, heure et lieu
  4. Ordre du jour : Rédigez l'ordre du jour détaillé
  5. Invitations : Le système invite automatiquement tous les adhérents actifs
📧 Étape 2 : Convocations automatiques

SAMA SYNDICAT envoie automatiquement :

  • Convocation initiale (15 jours avant)
  • Rappel automatique (3 jours avant)
  • Ordre du jour détaillé
  • Documents préparatoires
🎯 Étape 3 : Présidence de l'assemblée
Avant l'assemblée :
  • Vérifiez le quorum (affiché automatiquement)
  • Préparez votre discours d'ouverture
  • Révisez les points à l'ordre du jour
Pendant l'assemblée :
  • Marquez les présences en temps réel
  • Suivez l'ordre du jour
  • Lancez les votes électroniques
  • Enregistrez les décisions

🗳️ Gestion des Votes

En tant que président, vous contrôlez le processus de vote :

Processus de vote dans SAMA SYNDICAT : 1. Cliquez sur "Nouveau Vote" 2. Saisissez la question à voter 3. Choisissez le type de majorité requis 4. Lancez le vote électronique 5. Les résultats s'affichent en temps réel 6. Proclamez les résultats

📊 Suivi Post-Assemblée

Action Délai Responsable
Validation du PV 48h Président + Secrétaire
Diffusion du PV 1 semaine Secrétaire
Suivi des résolutions En continu Bureau exécutif

🤝 Conduite des Négociations

Préparation
  1. Analyse des données : Utilisez les statistiques SAMA SYNDICAT
  2. Définition des objectifs : Fixez des objectifs SMART
  3. Stratégie de négociation : Préparez votre argumentaire
  4. Équipe de négociation : Désignez les représentants
Pendant la négociation
  • Documentez chaque séance dans SAMA SYNDICAT
  • Enregistrez les positions de chaque partie
  • Notez les accords partiels
  • Planifiez les prochaines étapes

📋 Prise de Décision Stratégique

⚠️ Décisions importantes :
  • Lancement d'actions collectives
  • Acceptation/refus d'accords
  • Orientations stratégiques
  • Alliances avec d'autres syndicats
Processus de décision dans SAMA SYNDICAT :
  1. Analyse des données : Consultez les métriques
  2. Consultation : Sollicitez l'avis du bureau
  3. Vote si nécessaire : Organisez un vote électronique
  4. Communication : Informez les adhérents
  5. Suivi : Surveillez la mise en œuvre

🎤 Représentation Externe

Préparation des interventions :
  1. Briefing : Consultez les dossiers dans SAMA SYNDICAT
  2. Arguments : Préparez vos points clés
  3. Données : Munissez-vous des statistiques
  4. Mandats : Vérifiez vos pouvoirs de négociation
Suivi post-intervention :
  • Enregistrez les résultats dans le système
  • Informez le bureau exécutif
  • Communiquez aux adhérents si nécessaire
  • Planifiez les actions de suivi

🚨 Gestion de Crise

⚠️ Situations de crise :
  • Conflits sociaux majeurs
  • Médiatisation négative
  • Tensions internes
  • Urgences légales
Protocole de communication de crise :
  1. Évaluation rapide : Analysez la situation
  2. Cellule de crise : Réunissez le bureau
  3. Message unifié : Définissez la position officielle
  4. Communication : Utilisez tous les canaux SAMA SYNDICAT
  5. Suivi : Surveillez les réactions
